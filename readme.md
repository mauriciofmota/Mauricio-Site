# Core Portfolio & Interactive Architecture

A high-performance, minimalist developer portfolio and experimental sandbox environment built from scratch without bloated frameworks or AI-generated templates.

## 🛠️ Tech Stack & Systems
* **Interface Layer:** Native HTML5 and semantic, grid-driven CSS3.
* **Logic Core (Web):** Lightweight Vanilla JavaScript for local interface state management and high-performance canvas loops.

---

## ⚡ Key Engineering Decisions
* **Zero Frame-Stutter Design:** The core interactive profile layer matches raw CSS constraints for immediate page loading. No slow runtime UI libraries or modern CSS-in-JS overhead.

* **Non-Blocking Localization Engine:** Lightweight JavaScript inline parser reads native system parameters (navigator.languages) to handle dynamic English/Portuguese text state injection seamlessly before initial DOM painting.

* **Invisible DOM Overlay for Contact Fields:** The contact communication popup maintains active DOM presence to ensure native browser text search (Ctrl + F) mapping works flawlessly even when the modal is closed.
